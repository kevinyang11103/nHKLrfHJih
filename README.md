# 前言

欢迎来到本Spring Boot公司日常考勤系统项目！这是一个基于Java开发的实战项目，适用于毕业设计或学习实践。本文档将详细介绍项目内容、技术栈以及如何获取源码。希望这个项目能够对你有所帮助！

# 内容介绍

本项目是一款公司日常考勤系统，主要功能包括员工考勤记录、请假申请、加班申请等。系统采用前后端分离的设计模式，后端使用Spring Boot框架，前端使用Vue框架。通过这个项目，您可以掌握Java开发的基本技能，以及如何使用Spring Boot和Vue进行项目开发。
